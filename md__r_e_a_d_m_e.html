prerequisite

  1. install required packages by package system.
### debian/ubuntu
sudo apt install -y g++ gcc git doxygen libeigen3-dev libboost-all-dev libtinyxml-dev
  1. install some compiled dependency. There is a script to help you install these software.
# in lebai-motion-control
cd script
./install_prereqs.sh path_to_lmc_install
#path_to_lmc_install is a path to install lmc lib and include.

This script will try to download, compile and install some dependency to a pre-assigned location.

Compile and install

mkdir build
cd build
cmake .. -DCMAKE_INSTALL_PREFIX=path_to_lmc_install
make && make install

Unit test

In build binary directory run ctest

cd build
make
ctest

Document

In build biary directory, html directory is for the html document.

cd build
make doc
### doc will also be generated using just make

We now have an online document hosted by github page.

https://lebai-robotics.github.io/lmc_output/
