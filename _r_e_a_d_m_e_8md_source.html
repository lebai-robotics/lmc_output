3 ## prerequisite
4 
5 1. install required packages by package system.
6 
7 ```
8 ### debian/ubuntu
9 sudo apt install -y g++ gcc git doxygen libeigen3-dev libboost-all-dev libtinyxml-dev
10 ```
11 
12 
13 2. install some compiled dependency. There is a script to help you install these software.
14 
15 ```
16 # in lebai-motion-control
17 cd script
18 ./install_prereqs.sh path_to_lmc_install
19 #path_to_lmc_install is a path to install lmc lib and include.
20 ```
21 
22 This script will try to download, compile and install some dependency to a pre-assigned location.
