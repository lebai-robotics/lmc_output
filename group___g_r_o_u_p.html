◆ lmc_group_model_new()

lmc_group_model_t* lmc_group_model_new ( )

Create a group model instance.

Note
Group model is a data structure contains kinematic and model info. It's a model data structure contains model(kinematic, geometry shape, and etc...) information. These information will be used in collision detection and motion planner algortihm. Group model must be init by calling lmc_group_model_init_from_urdf_file before any usage.
