README

prerequisite

  1. install required packages by package system.
1 ### debian/ubuntu
2 sudo apt install -y g++ gcc git doxygen
  1. install some compiled dependency. There is a script to help you install these software.
1 # in lebai-motion-control
2 cd script
3 ./install_prereqs.sh install_dir
4 # install_dir is a path to install lmc lib
5 # Or for cross compile
6 './install_prereqs.sh install_dir target_platform gcc_major_version'
7 # target_platform can be ignored for the host platfrom."
8 # gcc_major_version should set match your host OS.

This script will try to download, compile and install some dependency to a pre-assigned location.

Compile and install

1 mkdir build
2 cd build
3 cmake .. -DCMAKE_INSTALL_PREFIX=path_to_lmc_install
4 make && make install

Unit test

In build binary directory run ctest

1 cd build
2 make
3 ctest

Document

In build biary directory, html directory is for the html document.

1 cd build
2 make doc
3 ### doc will also be generated using just make

We now have an online document hosted by github page.

https://lebai-robotics.github.io/lmc_output/
